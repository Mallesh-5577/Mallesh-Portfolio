# Mallesh Portfolio

Personal portfolio website of Mudavath Mallesh, focused on Full Stack Development.

## Overview

This portfolio highlights:

- Professional summary and contact details
- Technical skills and tools
- Education timeline
- Featured projects
- Certificates gallery with click-to-reveal animation and new-tab certificate preview

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Key Features

- Responsive two-column layout for desktop and single-column layout for mobile
- Smooth section navigation with active menu highlighting
- Project cards with clean presentation and external project link support
- Certificate section with:
  - Toggle button (show/hide)
  - Staggered reveal animations
  - Clickable cards opening full certificate in a new tab
- Accessibility improvements:
  - Better image alt text
  - Keyboard focus states
  - External link safety attributes
  - Reduced-motion support

## Run Locally

1. Clone or download the repository.
2. Open the project folder.
3. Launch [index.html](index.html) in any browser.

## Folder Structure

- [index.html](index.html): Main portfolio structure
- [style.css](style.css): Styling and animations
- certificates/: Certificate images
- logos/: Profile, skill, and project images
